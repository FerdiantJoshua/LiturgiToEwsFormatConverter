# Liturgi PDF to EasyWorship-ready Format Converter
By Ferdiant Joshua M.

## Requirements

1. Interpreter: Python 3.6
2. Packages
    ```shell script
    pip install -r requirements.txt    
    ```
   
## Run Program

1. Put liturgi files `.pdf` inside `input/`
2. Run:
    ```shell script
    python parse_pdf.py
    ```
   or
    ```shell script
    run_convert.bat
    ```
3. Resulting files will be inside `output/`
